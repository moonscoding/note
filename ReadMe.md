> 이미지관리방법

Typora는 clipboard 이미지를 불러올 수 있도록 지원합니다.

다만 등록한 이미지를 구글드라이브나 깃허브에 동기화하기 위해 

설정을 다음과 같이 바꿔야 합니다. (OSX)

- 편집 > 
- 이미지도구 > 
- 전역이미지설정... > 
- [이미지삽입에서] 현재폴더로 이미지 복사(./) 선택 > 
- 가능하다면 상대적 위치 사용 선택
